# MyBox收藏

## 编程相关

#### miktex

描述：Windows 平台上的 LaTeX 文档排版与编译环境。

```
scoop install main/miktex
```



#### mingw64

描述：用于在 Windows 系统上编译 C/C++ 等代码，提供类 Unix 开发环境的工具链集合。

```
scoop install main/mingw
```



#### Git

描述：分布式版本控制系统 Git 的 Windows 平台安装包。  

```
scoop install main/git
```



#### GitHub_CLI

描述：GitHub 官方命令行工具，用于在终端直接操作仓库和 PR。

```
scoop install main/gh
```



#### innosetup

描述：Inno Setup，一款流行的 Windows 程序安装包制作工具。

```
scoop install nonportable/innosetup-np
```



#### node.js

描述：Node.js JavaScript 运行时环境的 Windows 64位安装包。

```
scoop install main/nodejs
```



#### notepad4

描述：Notepad2 的增强版，轻量级代码编辑器，替代系统记事本。

```
scoop install extras/notepad4
```



#### pandoc

描述：文档格式转换神器，支持 Markdown、Word、HTML、PDF 等互转。

```
scoop install main/pandoc
```



#### python

描述：Python编程语言官方解释器及开发环境安装包。

```
scoop install main/python
```



#### vs_BuildTools

描述：Visual Studio 生成工具，提供 C++ 编译环境和 MSVC 工具链。



#### sublime text

描述：Sublime Text 4 代码编辑器的主程序文件。



#### typora

描述：一款优雅的**所见即所得 Markdown 编辑器**，无缝整合写作与预览界面，支持数学公式与多格式导出。



## 系统

#### rufus

描述：用于制作 USB 启动盘（如 Windows 安装盘）的轻量级工具。

```
scoop install extras/rufus
```



#### CrystalDiskInfo

描述：硬盘健康监测工具，用于读取 S.M.A.R.T. 信息判断硬盘寿命。

```
scoop install extras/crystaldiskinfo
```



#### Speccy

描述：Piriform 出品的系统硬件信息检测与规格查看工具。

```
scoop install extras/speccy
```



#### LTSC-Add-MicrosoftStore-master.zip

描述：为 Windows LTSC 企业版离线集成 Microsoft Store 的补丁包。



#### Microsoft-Activation-Scripts-master.rar

描述：基于 HWID/KMS38 的数字许可证激活脚本集合。



#### Microsoft-HEVCVideoExtension.zip

描述：Windows HEVC/H.265 视频编解码器扩展，用于播放高清视频。



#### DirectX Repair

描述：用于一键修复系统 DirectX 组件缺失导致的游戏无法运行问题。



#### VisualCppRedist_AIO_x86_x64.exe

描述：微软常用运行库合集，包含 VC++ 2005 至 2022 所有版本。



#### Win11Debloat.zip

描述：用于移除 Windows 11 预装应用、广告和隐私设置的 PowerShell 脚本。

```

```

#### spacesniffer
